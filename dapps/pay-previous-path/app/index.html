Update Path

Commands are like the SVG path d property only without commas: M, L, C, and Z commands only.
Values range from -128 to 127.

Setting the path will charge a fee. When the next person sets the path you can claim their fee (which may be less than the fee that you pay now) as long as you still control this address.
Do not change the gas price after you select "update" or else you will break the fee calculation and cause the transaction to fail!
